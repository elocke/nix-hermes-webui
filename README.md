# nix-hermes-webui

Nix flake packaging [`nesquena/hermes-webui`](https://github.com/nesquena/hermes-webui) —
a browser UI for the Hermes Agent. Provides a `python3.withPackages` derivation
and a `services.hermes-webui` NixOS module.

Status: pre-release, full README in v0.1.0.

See upstream feature request [#2422](https://github.com/nesquena/hermes-webui/issues/2422)
("nixos flake installer option") — this is one path toward that.
